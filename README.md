Numwords Example App
=========================

Example App using React, Redux & Express
<br/>

Install
-----
```
yarn install
```

Develop
-----
```
gulp serve
```
or
```
yarn run start
```

Build
-----
```
gulp build
```

Run ```gulp serve:prod``` to build and run the app

Test
-----
```
gulp test
```

Prettify
-----
```
gulp prettify
```
